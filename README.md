# RCREATORIX - Traditional Artist Portfolio

A premium, modern portfolio website for a traditional pencil and charcoal artist. This website showcases a dynamic and aesthetic UI/UX, designed to highlight hyper-realistic traditional artwork.

## Features

- **Modern UI/UX**: Built with vanilla HTML, CSS, and JavaScript.
- **Glassmorphism Design**: Aesthetic translucent overlays and blur effects.
- **Responsive Layout**: Optimized for both Desktop and Mobile devices.
- **Dynamic Instagram Feed**: Integration with Instagram Basic Display API to fetch and display the latest posts and reels automatically.
- **Masonry Gallery**: Image feed displayed in a 2-column stacked masonry layout (on mobile) and 4 columns (on desktop).
- **Custom Lightbox & Video Player**: Integrated Swiper.js for carousels and Plymouth (Plyr) for custom, scalable video playback of reels within a responsive lightbox.
- **Custom Cursor**: Interactive cursor and hover animations for an enhanced user experience.
- **Performance Optimized**: Lazy loaded images and smooth, lightweight animations using GSAP and ScrollTrigger.

## Tech Stack

- **HTML5**
- **CSS3** (CSS Variables, Flexbox, Grid, Media Queries)
- **JavaScript (ES6+)**
- **GSAP** (Animations)
- **Swiper.js** (Carousels and Sliders)
- **Plyr.js** (Video Player)
- **Three.js** (Background Particle Effects)

## Setup & Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser or serve it via a local development server (e.g., Live Server in VS Code, `python -m http.server`, etc.).
3. **Instagram API (Optional)**: To show your own Instagram feed, replace the placeholder `ACCESS_TOKEN` in `script.js` with your valid Instagram Basic Display API access token. If no token is provided, a setup banner is shown instead.

## Design Highlights

- **Hero Section**: Large typography with responsive text scaling and call-to-action buttons.
- **About Section**: Detailed biography layout with floating UI cards.
- **Process Slider**: A premium horizontal scrolling process walkthrough (Swiper integration).
- **Navigation**: Fixed glassy navigation with an animated hamburger menu for mobile devices.
